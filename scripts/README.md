# Scripts

Βοηθητικά scripts για εγκατάσταση και ρυθμίσεις.

- **install_ghostscript.bat** – Εγκατάσταση Ghostscript (απαιτείται για ορισμένες λειτουργίες PDF).
- **build_lite_from_kyria.py** – Μετά από αλλαγές στην `LOCAL_DEV/kyria/app_final.py`, αναγεννά την αυτόνομη `LOCAL_DEV/lite/app_lite.py` (χωρίς Streamlit πίνακες αποτελεσμάτων, κουμπί «Άνοιγμα / Προβολή», branding ATLAS Lite). Εκτέλεση από ρίζα repo: `python scripts/build_lite_from_kyria.py`.

Για εκκίνηση των εφαρμογών χρησιμοποιήστε το **run_app.bat** στη ρίζα του repo.
