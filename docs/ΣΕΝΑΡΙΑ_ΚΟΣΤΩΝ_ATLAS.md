# Σενάρια κόστους – ATLAS Lite

Όλες οι εκτιμήσεις είναι προσανατολιστικές. Τιμές σε USD (και περίπου σε EUR με ~1,08) εκτός αν αναφέρεται διαφορετικά.

Το project στοχεύει **Streamlit Community Cloud** (και προαιρετικά **δικό σας server / VPS**). Δεν χρησιμοποιείται Snowflake.

---

## Σενάριο χρήσης: 100 χρήστες/ημέρα, peak 11:00–13:00

| Παράμετρος | Τιμή |
|------------|------|
| **Χρήστες/ημέρα** | 100 |
| **Ταυτόχρονοι (peak)** | 40–50 |
| **Ώρα peak** | 11:00–13:00 (2 ώρες) |
| **Μέση διάρκεια σύνδεσης** | 15 λεπτά |

**Συμπέρασμα φόρτου:** Ο φόρτος συγκεντρώνεται σε 2 ώρες. Το **Community Cloud** έχει **resource limits**· για πολλούς ταυτόχρονους μπορεί να χρειαστείτε **ισχυρότερο hosting** (π.χ. VPS).

---

## 1. Streamlit Community Cloud (δωρεάν)

| Στοιχείο            | Τιμή   |
|---------------------|--------|
| **Μηνιαίο κόστος**  | **0 €** |
| **Σημείωση**        | Δεν προτείνεται για 40–50 ταυτόχρονους (peak)· κίνδυνος resource limits και αργής εφαρμογής. |

---

## 2. Self-hosted (VPS) – Streamlit στον δικό σας server

Τρέχετε μόνο τον server (Streamlit + Python). Καλό για 50–70 χρήστες αν έχετε 4–8 GB RAM και 2–4 vCPU.

| Πάροχος      | Πλάνο        | RAM   | vCPU | **EUR/μήνα (≈)** |
|--------------|--------------|-------|------|-------------------|
| Hetzner Cloud | CX31         | 8 GB  | 4    | ~12 €             |
| Hetzner Cloud | CX41         | 16 GB | 8    | ~21 €             |
| DigitalOcean | Basic        | 4 GB  | 2    | ~24 USD (~22 €)   |
| DigitalOcean | Basic        | 8 GB  | 4    | ~48 USD (~44 €)   |
| AWS Lightsail | 4 GB RAM     | 4 GB  | 2    | ~24 USD (~22 €)   |

**Εκτίμηση για ATLAS:** 8 GB RAM + 2–4 vCPU είναι λογικό target → **~12–25 €/μήνα** (Hetzner) ή **~22–44 €/μήνα** (DigitalOcean/Lightsail).

**Πλεονεκτήματα:** Χαμηλό κόστος, πλήρης έλεγχος, δεδομένα στο δικό σας περιβάλλον.  
**Μειονεκτήματα:** Χρειάζεται ρύθμιση, backups, ενημερώσεις.

---

## 3. Σύγκριση (peak 40–50 ταυτόχρονοι, 100 χρήστες/ημέρα)

| Λύση                 | Κατάλληλο για 50–70;     | Μηνιαίο κόστος (προσανατολιστικά) |
|----------------------|---------------------------|-------------------------------------|
| **Community Cloud**  | Όχι (resource limits)     | 0 €                                 |
| **Self-hosted VPS**  | Ναι (αν έχετε server)     | **~12–45 €/μήνα**                   |

---

## 4. Σύνοψη

- **Community Cloud:** δωρεάν, εύκολο deploy, **όχι** ιδανικό για πολλούς ταυτόχρονους στο peak.
- **Self-hosted (VPS):** πιο φθηνό για «βαριά» χρήση, με κόστος διαχείρισης server.
