================================================================================
ΟΔΗΓΟΣ: Εργασία πάνω στο CSS του παραγώμενου HTML και ενσωμάτωση αλλαγών
================================================================================

1. ΠΛΗΡΗΣ ΑΠΟΘΗΚΕΥΣΗ (full save)
--------------------------------------------------------------------------------
  • Ανοίξτε την εφαρμογή (Kyria ή Lite), φορτώστε PDF και κάντε επεξεργασία.
  • Πατήστε "Γρήγορη Προβολή - Lite" / "Προβολή αναφοράς" ώστε να ανοίξει
    το παραγόμενο HTML σε νέο παράθυρο.
  • Στο ανοιγμένο report, στο αριστερό sidebar υπάρχει κουμπί:
    "Πλήρης Αποθήκευση"
  • Πατήστε το: θα κατεβαίνει ένα αρχείο .html (π.χ. "Αναφορά - ATLAS Lite.html")
    με ΟΛΟ το τρέχον document (HTML + inline CSS + δεδομένα). Αυτό είναι η
    "πλήρης αποθήκευση".

2. ΔΟΥΛΕΙΑ ΠΑΝΩ ΣΤΟ ΑΠΟΘΗΚΕΥΜΕΝΟ ΑΡΧΕΙΟ
--------------------------------------------------------------------------------
  • Ανοίξτε το αποθηκευμένο .html σε επεξεργαστή κειμένου ή σε IDE.
  • Το CSS βρίσκεται μέσα στο αρχείο σε ένα ή περισσότερα <style>...</style>.
  • Κάντε τις αλλαγές που θέλετε στο CSS (χρώματα, αποστάσεις, γραμματοσειρές,
    sidebar, πίνακες, κ.λπ.).
  • Αποθηκεύστε το αρχείο και ελέγξτε το στο browser (άνοιγμα του αρχείου
    απευθείας) μέχρι να είστε ικανοποιημένοι.

3. ΠΟΥ ΕΝΣΩΜΑΤΩΝΟΥΜΕ ΤΙΣ ΑΛΛΑΓΕΣ ΣΤΟ ΠΡΑΓΜΑΤΙΚΟ ΠΑΡΑΓΩΜΕΝΟ
--------------------------------------------------------------------------------
  Το παραγόμενο HTML παράγεται από το module:

    html_viewer_builder.py

  • Το CSS του viewer (η σελίδα που ανοίγει με την "Προβολή") ορίζεται στην
    σταθερά VIEWER_STYLES (περίπου γραμμή 1729 και μετά).

  • Για να ενσωματώσετε τις αλλαγές:
    1. Ανοίξτε το html_viewer_builder.py.
    2. Βρείτε τη γραμμή:   VIEWER_STYLES = """
    3. Αντικαταστήστε το περιεχόμενο της VIEWER_STYLES με το νέο CSS που
       έχετε στο αποθηκευμένο .html (μόνο το περιεχόμενο ΜΕΣΑ στο <style>...</style>,
       χωρίς τα tags <style> και </style>).
    4. Προσοχή: η σταθερά τελειώνει με """ (τριπλά εισαγωγικά). Μην σβήσετε
       το κλείσιμο """ στο τέλος.

  • Αν έχετε αλλάξει και δομή HTML (classes, ids) στο αποθηκευμένο αρχείο,
    τότε εκτός από VIEWER_STYLES ίσως χρειαστεί να τροποποιήσετε και το
    template στο build_viewer_html_document (περίπου γραμμές 1486–1561), όπου
    συναρμολογείται το πλήρες HTML document.

4. ΕΚΤΥΠΩΣΗ (print)
--------------------------------------------------------------------------------
  Αν θέλετε να αλλάξετε και το CSS της εκτυπώσιμης μορφής (που ανοίγει με
  "Εκτύπωση"), οι κανόνες βρίσκονται στην σταθερά PRINT_STYLES στο ίδιο
  αρχείο (html_viewer_builder.py, πριν την VIEWER_STYLES). Η ροή είναι η ίδια:
  αποθήκευση → επεξεργασία στο saved HTML (αν η print view είναι μέσα) →
  αντιγραφή του σχετικού CSS στο PRINT_STYLES.

5. ΣΥΝΤΟΜΗ ΑΝΑΦΟΡΑ
--------------------------------------------------------------------------------
  Αρχείο:        html_viewer_builder.py
  Viewer CSS:    VIEWER_STYLES (περίπου γραμμή 1729)
  Print CSS:     PRINT_STYLES (πριν την VIEWER_STYLES)
  Template:      build_viewer_html_document() — χρησιμοποιεί <style>{VIEWER_STYLES}</style>
